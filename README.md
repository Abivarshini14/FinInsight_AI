💰 FinInsight AI – Personal Finance Assistant

Overview
      
      FinInsight AI is an intelligent financial management web application built using Streamlit, Google Gemini AI, and Plotly.
      The application helps users analyze their financial health, visualize spending patterns, calculate investment returns, and receive AI-powered financial guidance.

Features 

🏠 Financial Dashboard

* Track monthly income and expenses
* Calculate net savings and savings rate
* View a Savings Health Score using an interactive gauge chart
* Get personalized financial insights and recommendations
* Emergency fund target estimation
 🤖 AI Financial Bot

* Powered by Google Gemini AI
* Answers finance-related questions
* Provides guidance on:

  * Investments
  * SIP Planning
  * Budgeting
  * Tax Basics
  * Personal Finance Management
* Includes retry logic for API quota handling

 🧮 Financial Calculators
 
 SIP Calculator

* Calculate future value of Systematic Investment Plans
* Estimate long-term wealth creation

EMI Calculator

* Calculate monthly loan repayments
* Supports customizable loan amounts, interest rates, and tenure

📊 Budget Visualizer

* Interactive expense tracking
* Pie chart visualization of spending categories
* Analyze expense distribution across:

  * Rent/EMI
  * Food
  * Bills
  * Other Expenses
